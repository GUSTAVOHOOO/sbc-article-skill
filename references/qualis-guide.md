## O QUE É O QUALIS CAPES

O QUALIS é o sistema oficial brasileiro de estratificação de produção científica,
usado por programas de pós-graduação para avaliar a qualidade das publicações.

**Estratos atuais (Ciência da Computação):**
- **A1** — máximo; impacto internacional alto
- **A2** — muito bom; conferências/periódicos top da área
- **A3/A4** — bom; conferências nacionais consolidadas
- **B1-B4** — regular; eventos menores ou regionais
- **C** — sem pontuação relevante para pós-graduação

**Como encontrar o QUALIS de um venue:**
1. Acesse: https://sucupira.capes.gov.br/sucupira/public/consultas/coleta/veiculoPublicacaoQualis/listaConsultaGeralPeriodicos.jsf
2. Busque pelo ISSN, título ou sigla do evento
3. Filtre por área "CIÊNCIA DA COMPUTAÇÃO"

## COMO SELECIONAR O VENUE CERTO

### Passo 1: Identifique a área primária
- Se o trabalho tem experimentos de ML → IA/ML
- Se o trabalho é sobre sistema/arquitetura → ES ou Sistemas
- Se o trabalho envolve dados/consultas → BD/RI
- Se o trabalho tem componente de educação → Educação

### Passo 2: Avalie o nível do trabalho
| Característica | Venue recomendado |
|---|---|
| Contribuição metodológica nova, avaliação robusta | BRACIS, SBBD, SBES, SBRC (A2/A3) |
| Aplicação em novo domínio com análise | ENIAC, KDMILE, IHC (B1/B2) |
| Trabalho em andamento / preliminar | Workshops e trilhas WIP de qualquer evento |
| Resultado de TCC / iniciação científica | ERCAS, WUE, workshops de alunos |

### Passo 3: Verifique compatibilidade de escopo
Leia o **CFP (Call for Papers)** do evento e confira:
- Os tópicos de interesse contemplam sua área?
- O tipo de contribuição é aceito (teórico, experimental, survey, ferramenta)?
- O tamanho de página é adequado para o trabalho?

### Passo 4: Verifique deadlines
Calendário geral (aproximado — confirme no site do evento):
- **Jan-Mar:** Submissões para eventos de meio do ano (SBRC, CSBC)
- **Abr-Jun:** Submissões para eventos de agosto/setembro (CBSoft, SBES)
- **Jul-Set:** Submissões para eventos de final do ano (SBBD, BRACIS, IHC, SBIE)

## ESTRATÉGIA DE SUBMISSÃO

### Estratégia conservadora (garantir publicação):
1. Submeta para evento B1/B2 na área
2. Use o feedback para fortalecer o trabalho
3. Submeta versão expandida para periódico ou evento A2/A3

### Estratégia agressiva (maximizar impacto):
1. Vise diretamente BRACIS, SBBD ou SBES
2. Prepare para rejeição com feedback valioso
3. Revise e resubmeta melhorado

### Regra de ouro:
**Nunca submeta o mesmo artigo simultaneamente para dois eventos.**
Submissão simultânea é antiética e pode resultar em banimento dos autores.
